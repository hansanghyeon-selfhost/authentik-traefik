
## authentik

authentik란?

인증을 아주 간편하게 만들어줍니다.

왜 이것을 사용하는가? 무수히 selfhost 서비스들이있고 각 서비스마다 모두 auth 기능을 추가할수없습니다. authentik를 통해서 인증을 아주 간편하게 넣을 수 있습니다.
